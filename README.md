# 普高达集团 - 尊享生活体验中心

一个华丽的、高端的、响应式的网站，展示普高达集团的多个展厅和服务。

## 📁 项目结构

```
Pyramid-Flow-main/
├── index.html       # 主页面
├── styles.css       # 样式表（包含所有响应式设计）
├── script.js        # JavaScript交互脚本
└── README.md        # 本文件
```

## ✨ 功能特点

### 1. **导航系统**
- 固定式导航栏，支持平滑滚动
- 响应式汉堡菜单（手机端）
- 自动跟踪当前活跃章节

### 2. **首页 Hero Section**
- 全屏视频背景
- 渐变色叠加效果
- 滚动指示器动画
- 响应式标题文字

### 3. **展厅导航网格**
- 7个展厅卡片
- 悬停动画和变换效果
- 支持点击跳转到对应展厅
- 彩色渐变背景

### 4. **展厅详情页面**
- **大健康产品展厅**：健康产品展示与分类
- **海珍品展厅**：海洋食材精选
- **山珍品展厅**：山林特产展示
- **墨客小院**：九宫格文化艺术中心
  - 中心为主题，周围8个子项目围绕
  - 悬停时图标旋转动画
- **医疗中心**：6个医疗服务卡片
- **非遗中心**：4个文化传承项目
- **白金卡**：尊贵会员卡展示
  - 3D卡片效果
  - 专享权益列表

### 5. **视觉效果**
- ✨ 鼠标跟随粒子效果
- 🎨 多个梯度渐变色
- 🔄 平滑过渡动画
- ⚡ 悬停放大/变换
- 📱 完全响应式设计

### 6. **交互功能**
- 平滑页面滚动
- 表单验证和提交
- 通知提示系统
- 视差滚动效果

### 7. **性能优化**
- 防抖和节流函数
- 交叉观察器用于动画触发
- 页面隐藏时暂停动画

## 🎨 设计特点

### 色彩方案
- **主色**：金色（#d4a574）- 代表尊贵和高端
- **辅色**：深灰（#1a1a1a）- 代表专业和稳重
- **强调色**：浅金（#f0e68c）- 代表活力和高级

### 响应式断点
- **桌面**：1200px以上
- **平板**：768px - 1024px
- **手机**：480px - 768px
- **超小屏**：480px以下

## 🚀 快速开始

### 1. **在浏览器中打开**
直接在浏览器中打开 `index.html` 文件即可查看。

### 2. **使用本地服务器（推荐）**
```bash
# 使用Python
python -m http.server 8000

# 或使用Node.js的http-server
npx http-server

# 或使用VS Code的Live Server扩展
```

访问 `http://localhost:8000` 查看网站。

## 🔧 高级功能（可选）

### 启用粒子效果
在 `script.js` 的最后取消注释：
```javascript
initParticles(); // 鼠标移动时产生粒子
```

### 修改粒子颜色
在 `initParticles()` 函数中修改 `color` 值：
```javascript
this.color = `rgba(212, 165, 116, ${Math.random() * 0.5})`;
```

### 自定义通知样式
修改 `showNotification()` 函数中的样式配置。
